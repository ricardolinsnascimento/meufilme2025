Wiki Oficial do Balatro

O jogo de cartas viciante!

Sinopse: Balatro é um jogo indie de construção de baralho com mecânicas inspiradas no pôquer. O jogador monta mãos para pontuar, usando cartas especiais, coringas, cartas de tarô, planetas e multiplicadores. O desafio está em criar combinações únicas a cada rodada para progredir nos níveis.
